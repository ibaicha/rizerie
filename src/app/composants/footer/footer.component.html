A propos

Basé à Saint-Louis, le cabinet SIS’Tech intervient dans le développement de solutions informatiques spécifiques, notamment le développement de logiciels (client lourd), le développement d’application web, le développement mobile, la conception de base de données etc. Entre autres activités informatiques, SIS’Tech est aussi contracté dans le conseil, l’infogérance, l’assistance et la formation des utilisateurs, etc.
